# Shoppepay-Callback

Layanan verifikasi dan monitoring transaksi QRIS ShopeePay Merchant secara otomatis menggunakan Web Partner API.

## Fitur
*   Scraping transaksi otomatis dari ShopeePay Partner Portal.
*   Webhook callback instan ke sistem Next.js saat pembayaran terdeteksi.
*   Log mutasi & pencocokan transaksi aman.
